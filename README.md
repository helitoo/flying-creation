# Flying creation

My portfolio website built with **React**, **TypeScript**, **Vite**, **Three.js**, **React Three Fiber**, **GSAP**, and **Tailwind CSS**.

The website showcases my work experience, personal works and contact information with smooth scroll-based animations and an immersive 3D experience.

## Tech Stack

- React 19
- TypeScript
- Vite
- Three.js
- React Three Fiber
- React Three Drei
- GSAP
- Tailwind CSS

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
